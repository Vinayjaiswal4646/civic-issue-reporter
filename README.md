# Civic Issue Reporting App 🚀

A web application where users can report civic issues like garbage and potholes with image upload and real-time GPS location.

## 🔥 Features

- Upload issue with image 📸  
- Automatic GPS location detection 📍  
- Interactive map with markers 🗺️  
- Data stored using SQLite 💾  

## 🛠️ Tech Stack

- Python (Flask)  
- HTML, CSS, JavaScript  
- Leaflet.js  
- SQLite  

## ▶️ How to Run

```bash
pip install flask
python app.py
